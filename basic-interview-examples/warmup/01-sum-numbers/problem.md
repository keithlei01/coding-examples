# 01 — Sum Numbers

Warm up loops and arrays.

## Task

Implement `sumNumbers(nums)` that returns the sum of all numbers in the array. Empty array → `0`.

## Examples

```javascript
sumNumbers([10, 20, 5]); // 35
sumNumbers([]);            // 0
sumNumbers([-2, 2]);       // 0
```

## Constraints

- `nums` contains only numbers.
- Length ≤ 10_000.

## Hint

Either a `for` loop or `reduce` — both are fine.
