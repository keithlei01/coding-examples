# 10 — Unique In Order

Warm up deduplication while preserving order — e.g. unique users in a session trail.

## Task

Implement `uniqueInOrder(arr)` returning a new array with duplicates removed, keeping the **first** occurrence of each value.

## Examples

```javascript
uniqueInOrder(["a", "b", "a", "c", "b"]); // ["a", "b", "c"]
uniqueInOrder([1, 1, 2, 1]);              // [1, 2]
uniqueInOrder([]);                        // []
```

## Constraints

- Length ≤ 20_000.
