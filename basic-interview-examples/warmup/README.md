# Warm-Up Exercises (10)

**Start here** if you have not coded in a while. Same layout as easy / medium / hard:

| Folder | You do | Answer (after you try) |
|--------|--------|-------------------------|
| `01-…/problem.md` | Read & implement | `solution.js`, `solution-modern.js`, `explanation.md` |

Do **not** open the solution files until you have a working attempt.

| File | Style |
|------|--------|
| `solution.js` | **Classic** — loops, plain objects, step-by-step |
| `solution-modern.js` | **Modern** — `reduce`, `filter`, `Map`, `Set`, etc. |

**Pace:** ~15–20 min per exercise → 2–3 days, then [easy/](../easy/).

| # | Exercise |
|---|----------|
| [01-sum-numbers](./01-sum-numbers/) | Loop / reduce |
| [02-count-occurrences](./02-count-occurrences/) | `Map` counts |
| [03-group-by-key](./03-group-by-key/) | Group records |

| [10-unique-in-order](./10-unique-in-order/) | Dedupe in order |

**Practice without hints:** this repo disables inline suggestions in [.vscode/settings.json](../.vscode/settings.json). Also turn off **Cursor Tab** (see below).

Each folder has **`attempt.js`** ready — fill in the `TODO`, then run:

```bash
node warmup/01-sum-numbers/attempt.js
```
